After installing the FPTN client, you can find its icon in the system tray. The tray icon provides quick access to all essential features of the VPN client. Simply click on the icon to open the context menu, where you can access the settings, add or manage servers, and connect or disconnect from the VPN server with ease.

Registering for access to free internet through FPTN is very simple! Use our Telegram @fptn_bot, which will quickly provide you with server information as well as a login and password for free internet access.

Right-click on this icon to open a context menu where you can access the settings. In the settings window, you can configure the client by adding a new server.
